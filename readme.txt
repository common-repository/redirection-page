== Description ==

Redirect your specified pages, it is useful when you have 404/not-found pages.

* <a href="http://www.yusuf.asia/go/p4-homepage/">Plugin</a> by <a href="http://www.yusuf.asia/">Yusuf</a>

== Installation ==

1. Upload 'redirection-page' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your specified pages in "Settings Page" menu
